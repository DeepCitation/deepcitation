Brand Iteration Considerations

Internal note for extending the DeepCitation brand language without collapsing the SDK into the full web-app palette.

The anchor-text highlight yellow is now a deliberate special accent in the brand system. It is not a general-purpose CTA color. It exists to signal a semantic distinction: the verified anchor is narrower than the full verified phrase.

Special Accent

Accent Value Usage Constraint
anchor-highlight rgba(251, 191, 36, 0.2) Anchor text highlight in light proof contexts Use only when anchor text differs from the broader verified phrase
anchor-highlight-dark rgba(251, 191, 36, 0.25) Anchor text highlight in dark proof contexts Preserve contrast without turning the accent into a general warning color

Guidance

  • Keep paper-white and cream as the warm editorial base for first-party branded shells.
  • Use the slate ramp for typography, borders, and structure, not for flattening the interface into cold grayscale.
  • Reserve the amber anchor highlight for evidence semantics so it reads as a distinctive DeepCitation signature.
  • Do not reuse the anchor highlight yellow for primary actions, warnings, or generic decorative emphasis.
  • In SDK surfaces, preserve host override flexibility even when a first-party branded wrapper uses these reference tokens.
